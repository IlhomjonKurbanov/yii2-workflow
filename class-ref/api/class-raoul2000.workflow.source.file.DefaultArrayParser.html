Class DefaultArrayParser

Implements a parser for default PHP array formatted workflow.

Example :

[
  'initialStatusId' => 'draft',
  'status' => [
      'draft'     => [
          'label'      => 'Draft State'
          'transition' => 'published'
      ],
      'published' => [
          'transition' => ['draft','published']
      ],
      'archived'
  ]
]
